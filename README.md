# Homebrew Tap for Nexis

[Nexis](https://github.com/s4solutionsllc/Nexis) — macOS System Optimizer and Monitoring.

## Install

```bash
brew tap s4solutionsllc/nexis
brew install --cask nexis
```

## Upgrade

```bash
brew upgrade --cask nexis
```

## Uninstall

```bash
brew uninstall --cask nexis
brew untap s4solutionsllc/nexis
```

## Requirements

- macOS 11 (Big Sur) or later
- Apple Silicon (M1/M2/M3/M4)
